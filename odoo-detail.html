  • System Overview

    Odoo is an integrated Enterprise Resource Planning (ERP) system that helps organizations manage and automate core business processes such as Sales, CRM, Inventory, Accounting, HR, and Project Management in one centralized platform. Its modular and scalable design allows businesses to customize the system based on operational needs, improving efficiency, data accuracy, and overall productivity.

  • Role

    System Testing

    • Performing functional and regression testing.
    • Identifying and documenting defects, and validating fixes.
    • Support User Acceptance Testing (UAT) to ensure that Odoo features and workflows meet business requirements before deployment.

    System Supporting
    • Setting up Odoo systems for small projects, including configuration, testing, and deployment to production.
    • Ensuring the smooth daily operation of the Odoo system.
    • Providing user support, troubleshooting system and functional issues, managing user access and permissions, assisting with system configurations, and supporting system updates.
    • Work closely with users and technical teams to resolve issues quickly and maintain system reliability.
