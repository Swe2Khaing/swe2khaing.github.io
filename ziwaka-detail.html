PORTFOLIO

Specialist Clinic System using Apriori Algorithm

  • Duration

    2022 ~ 2023

  • Organization

    University of Computer Studies-Sittway, Rakhine, Myanmar

  • System Overview

    Ziwaka Specialist Clinic System is an online web-based system created to upgrade from the previous manual booking system where patients have to come in person. This system also helps manage data systematically, which drastically improves data bloat and storage errors.

    Traditional data analysis methods often fail to uncover hidden relationships among symptoms, diagnoses, tests, and specialist referrals. Therefore we have used Apriori Algorithm in this system to discover hidden patterns and associations to improve clinical decision-making, resource allocation, and patient care.

  • Role

    As a member of the project team, I was actively involved in both the design and development across all phases of the project lifecycle as part of my graduation project.
